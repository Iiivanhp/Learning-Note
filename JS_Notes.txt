Javascript Notes
1.基本语法
	语句以；结束
	语句块用{}包裹
	基础赋值语句 var x=1;
	注释符 //  或者 /* */
	严格区分大小写
2.数据类型
	number 数值类型不区分整数和浮点数
		NaN  无法表示，not a number
		Infinity 无限大
		十六进制数 0xf22a
	
	字符串 '' 或者 ""
	布尔值
	
	比较运算符 == 和 ===
	==比较符会自动转换数据类型，===则不会
	
	null 空值 	undefined 未定义（通常用于判断函数参数是否传递）
	
	数组 
		元素可以是任意类型，只需用[,]表示
		也可以通过Array()函数实现
		索引（下标）从0开始
	
	变量
		变量声明用 var语句
		js是动态语言，变量类型不固定
		strict模式 'use strict'	强制通过var声明变量
3.字符串
	转义符\
	用\表示ASCII字符和Unicode字符
	反引号`` 用于表示多行字符串
	模板字符串 用${变量名}替换字符串
	
	字符串操作
	字符串具有索引，从0开始	
	字符串本身不可改变
	相关函数 length toUpperCase indexOf substring
